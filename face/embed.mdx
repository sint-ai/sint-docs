# Embedding Face

Three ways to put Face in your app.

## Path 1: Web component (easiest)

```html
<script src="https://face.sint.gg/embed.js" async></script>

<sint-face
  agent-endpoint="https://your-agent.example.com/v1/chat"
  voice="alloy"
  policy-gateway="https://gate.your-tenant.sint.gg"
  capability-token="..."
  theme="dark"
></sint-face>
```

Drop into any HTML; renders a self-contained Face. Communicates with your agent endpoint over WebSocket; submits actions to your Policy Gateway.

## Path 2: React component

```bash
npm install @sint/face-react
```

```tsx
import { SintFace } from "@sint/face-react";

<SintFace
  agentEndpoint="https://your-agent.example.com/v1/chat"
  voice="alloy"
  policyGateway="https://gate.your-tenant.sint.gg"
  capabilityToken={token}
  theme="dark"
  onWidgetSpawn={(w) => { /* custom render hook */ }}
  onApproval={(a) => { /* custom approval handler */ }}
/>
```

## Path 3: Iframe

```html
<iframe
  src="https://face.sint.gg/embed?agent=https%3A%2F%2Fyour-agent...&token=..."
  allow="microphone; camera; autoplay"
  style="width:100%; height:600px; border:0;"
></iframe>
```

Lowest-touch; useful when you can't add scripts to the host page. Limited to Face's default styling.

## Required configuration

- `agent-endpoint` — your agent's WebSocket or streaming HTTP endpoint
- `policy-gateway` — your Policy Gateway base URL
- `capability-token` — the token Face will attach to actions; should be scoped narrowly to what Face is allowed to do on the user's behalf

Optional:

- `voice` — ElevenLabs voice ID (default: a neutral voice we ship)
- `theme` — `dark` (default) | `light`
- `widgets` — list of allowed Widget Layer component types
- `consent` — `required` (default) | `assumed` — whether to surface a consent prompt before mic access

## Permissions

The host page must grant microphone access. Face requests it on first interaction. Camera is optional; only required if you enable lip-sync verification (a future feature).

## SDK

The full Face SDK is at [`github.com/sint-ai/face-sdk`](https://github.com/sint-ai/face-sdk) (private; access on request). The SDK lets you customize the avatar, register custom Widget Layer components, and hook into every event in the pipeline.
