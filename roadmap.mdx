## What's explicitly not on the roadmap

We've said "no" to several directions. For transparency:

- **$SINT token revival** — archived indefinitely. If we revisit, it will be a separate brand with a separate announcement.
- **Consumer-facing agent marketplace** — not our focus. The protocol enables it; we don't build it.
- **LLM hosting** — we govern agents running against any LLM; we don't host models.
- **General-purpose no-code agent builder** — SINT Console has a visual workflow builder for SINT-stack agents; we don't compete with n8n or Zapier.
