# SINT Labs

This is the engineering documentation for SINT Labs (PSHKV Inc., Los Angeles).

If you're looking for the company and its services, start at **[sint.gg](https://sint.gg)**.

If you're looking to build with the Protocol, install Operators, or integrate the Platform — you're in the right place.

<Note>
**A note on the past.** SINT pivoted away from its earlier Web3/agent-marketplace direction in early 2026. SINT Labs does not issue, sell, or plan to issue a token. Read the full pivot story at [/origin](/origin).
</Note>
