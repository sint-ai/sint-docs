# Quickstart

There's no single quickstart for SINT because there's no single thing to do with it. Pick the path.

## I want to run the Protocol locally

The fastest way to see the safety envelope in action.

```bash
# Prerequisites: Node.js >= 22, pnpm >= 9
git clone https://github.com/sint-ai/sint-protocol
cd sint-protocol
pnpm install
pnpm run build
pnpm run test                  # full workspace test suite

# Start the Policy Gateway
pnpm --filter @sint/gateway-server dev
#   → http://localhost:3100/v1/health
#   → http://localhost:3100/v1/ready
#   → http://localhost:3100/v1/docs

# Or start the full stack: Gateway + Interface + Postgres + Redis
pnpm run stack:interface
#   → http://localhost:3202
```

Other stack profiles for specific scenarios:

```bash
pnpm run stack:dev                       # local dev with hot-reload
pnpm run stack:edge                      # edge-deployable profile
pnpm run stack:prod-lite                 # production-shape, single host
pnpm run stack:gazebo-validation         # robotics simulation harness
pnpm run stack:isaac-sim-validation      # NVIDIA Isaac Sim integration
```

Continue at [Protocol Quickstart](/protocol/quickstart) for a guided tour.
