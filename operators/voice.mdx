# Voice Operator

**One-liner:** Handles the calls your team shouldn't be spending time on — intake, scheduling, confirmation, triage — and ledgers every decision.

## What it replaces

- Front-desk time on scheduling and confirmations
- After-hours voicemail-to-callback loops
- Call center tier-1 intake for qualifying pre-triage scenarios

## System

```
PSTN / SIP
   ▼
ASR (streaming)
   ▼
Agent plan
   ▼
Gate check
   ▼
TTS (streaming)
   ▼
Action (calendar / CRM / ticket)
   ▼
Ledger
   ▼
Handoff (when required)
```

## Safety envelope

| Tier | Action classes |
|---|---|
| T0 | `call.read`, `intent.classify` |
| T1 | `calendar.offer-slots`, `calendar.book`, `confirm.appointment` |
| T2 | `calendar.reschedule`, `calendar.cancel` |
| T3 | Any billing or payment reference |
| T4 | Any clinical or legal triage (always handoff) |

Hard limits:

- PHI/PII never leaves tenant namespace
- HIPAA BAA with LLM provider required before deploy
- Configurable call recording retention (default 30 days)
- No outbound call without explicit pre-registered campaign authorization

## Metrics

- Call completion rate
- Average handle time
- Schedule accuracy (vs follow-up confirmation)
- Handoff precision (false positive / false negative against audit)
- Post-call CSAT
- Human-hours reclaimed
