## Highlights

### Q1 2026

- **Protocol v0.8** — first release under the rebranded SINT Labs; clarified Economy is optional; property test coverage raised across all five invariants; policy hot-reload landed
- **Canon v0.4** — content-addressed manifest, per-scope audit, deterministic retrieval as default
- **Face v0.5** — streaming TTS pipeline; 340ms median TTFA (from 780ms in v0.4); Widget Layer registration API
- **SOP beta** — internal beta with four Agency clients; YAML Pipelines compile + deploy path stable
- **ROSClaw** — arXiv preprint (2603.26997); IROS 2026 submission accepted
- **MCP Security Analysis** — arXiv preprint (2601.17549)
